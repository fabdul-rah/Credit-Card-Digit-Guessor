Luhn check digit

Enter every digit except the final check digit. The correct last digit is computed in your browser using the same mod-10 rule used by Visa, Mastercard, AmEx, and other issuers.

What this project shows

  • Algorithms — correct Luhn parity for any body length (16-digit Visa-style and 15-digit AmEx-style).
  • Static web — this Pages build is plain HTML, CSS, and ES modules (no server); the repo also includes Python + Flask for local demos and pytest.
  • Quality — JavaScript mirrors luhn.py; run pytest in the repository to verify the reference implementation.
  • Digital citizenship — checksums do not prove a card is real, funded, or yours; use test numbers only.
