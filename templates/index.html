Luhn check digit

Enter every digit except the final check digit. The correct last digit is computed using the same mod-10 rule used by Visa, Mastercard, AmEx, and other issuers.

What this project shows

  • Algorithms — correct Luhn parity for any body length (16-digit Visa-style and 15-digit AmEx-style).
  • Full stack — Python + Flask API, semantic HTML, responsive CSS, and progressive enhancement with JavaScript.
  • Quality — automated tests (pytest) and a JSON API suitable for demos or grading rubrics.
  • Digital citizenship — clear warnings: checksums do not prove a card is real, funded, or yours; use test numbers only.
